 NFA to DFA Simulator

This project is a simple web application that shows how a Non-Deterministic Finite Automaton (NFA) can be converted into a Deterministic Finite Automaton (DFA).

The main idea is to understand subset construction in an easy and visual way. Along with conversion, the project also allows testing input strings on the generated DFA

 This project allows you to select an NFA
- Converts it into an equivalent DFA
- Shows the step-by-step conversion process
- Displays the DFA transition table
- Allows you to test input strings
- Shows whether the string is accepted or rejected

 Why predefined NFAs are used

In this project, a few NFAs are already defined instead of taking custom input from the user.

This is done because:
- It avoids invalid or incorrect inputs
- It keeps the focus on understanding the conversion process
- It makes the simulation faster and easier to use
- It ensures that all examples work correctly

The selected NFAs are simple and cover common patterns like:
- strings ending with a pattern
- strings containing a substring
- checking even number of symbols

 How to use

1. Select any NFA from the given options
2. Click on "Convert to DFA"
3. Observe how the DFA is constructed step by step
4. Enter a binary string
5. Click on "Test String"
6. Check whether the string is accepted or rejected
